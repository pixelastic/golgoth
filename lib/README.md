# golgoth

<div class="lead">
  When building project in JavaScript, I often end up installing the same
  dependencies over and over again. Golgoth is a meta-package that allow me to
  install them all in one go.
</div>

## Libraries included

| Library name       | Required  as          | Description                                                                            |
| ------------------ | --------------------- | -------------------------------------------------------------------------------------- |
| [dayjs][1]         | `golgoth/dayjs`       | 2KB immutable date time library alternative to Moment.js with the same modern API      |
| [got][2]           | `golgoth/got`         | Human-friendly and powerful HTTP request library for Node.js                           |
| [lodash][3]        | `golgoth/_`           | A modern JavaScript utility library delivering modularity, performance & extras.       |
| [pify][4]          | `golgoth/pify`        | Promisify a callback-style function                                                    |
| [p-all][5]         | `golgoth/pAll`        | Run promise-returning & async functions concurrently with optional limited concurrency |
| [p-map][6]         | `golgoth/pMap`        | Map over promises concurrently (use `{concurrency: 1}` for serial execution)           |
| [p-props][7]       | `golgoth/pProps`      | Like Promise.all() but for Map and Object                                              |
| [query-string][8]  | `golgoth/queryString` | Parse and stringify URL query strings                                                  |
| [time-span][9]     | `golgoth/timeSpan`    | Simplified high resolution timing                                                      |
| [yoctocolors][10]  | `golgoth/yoctocolors` | The smallest and fastest command-line coloring package on the internet                 |

## Usage

```javascript
const pMap = require('golgoth/pMap');
const _ = require('golgoth/lodash');
```
